This is a starter template for [Learn Next.js](https://nextjs.org/learn).

## Setting Up
### Config Eslint
Add script to `package.json`
```
"script": {
    "lint": "next lint"
}
```

Running script and install dependencies
```
yarn lint OR npm lint
```

Install lint-staged
```
https://github.com/okonet/lint-staged
```
